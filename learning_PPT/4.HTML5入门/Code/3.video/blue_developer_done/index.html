我们是谁？

电子科大·开发者联盟是由电子科大计算机协会创办的学生群体开发者社区，由于社区自由、开放，高手云集，吸引了来自各个学院、不同年级的同学加入到这个群体中来，同时，协会也对外接收项目，为有需要的同学提供服务~
